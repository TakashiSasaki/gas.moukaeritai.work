What's this

Javascript code in Github can't be used in browsers directly because its Content-Type is "text/plain" in raw mode. This Google Apps Script web service fetches and emits them with "text/javascript" MIME type. Fetched script will be cached for up to 21600 seconds (6 hours) if the size is smaller than 100KB approx.

How to use

<script src="https://script.google.com/macros/s/AKfycbwu69ThF572u9fRQmN0CC8P6g0ymZ1gUtQWQRleNIBJSF1IKms/exec?Script Name" async></script>
